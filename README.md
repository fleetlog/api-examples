#Fleetlog oAuth2 example

### Install required modules
```
    npm install
```

### Run the app
```
    node app FLEETLOG_CLIENT_ID=<YOUR_CLIENT_ID> FLEETLOG_CLIENT_SECRET=<YOUR_CLIENT_SECRET>
```

Open `localhost:3001` in your browser.
